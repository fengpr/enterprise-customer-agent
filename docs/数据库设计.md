# 数据库设计

## 核心实体

### user_account

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| id | BIGINT | 主键 |
| username | VARCHAR(64) | 登录名 |
| password_hash | VARCHAR(255) | 密码摘要 |
| real_name | VARCHAR(64) | 姓名 |
| role_code | VARCHAR(32) | 角色编码 |
| team_id | BIGINT | 所属团队 |
| status | TINYINT | 0 禁用，1 启用 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## 建议索引

- `order_info(order_no)` 唯一索引。
- `logistics_info(order_no)` 唯一索引，用于按订单号查询物流主记录。
- `logistics_trace(logistics_id, occurred_at)` 轨迹时间索引，用于按物流单展示完整路线。
- `support_ticket(ticket_no)` 唯一索引。
- `support_ticket(status, priority, sla_deadline)` 工单队列索引。
- `ticket_urge_log(ticket_no, created_at)` 工单催办历史索引。
- `chat_session(session_no)` 唯一索引。
- `chat_message(session_id, created_at)` 会话消息索引。
- `knowledge_document(status, doc_type, effective_time, expire_time)` 知识库检索过滤索引。
- `agent_call_log(session_id, created_at)` Agent 审计索引。
